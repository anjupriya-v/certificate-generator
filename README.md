## Procedure to run this application
- Clone the repository
```
$ git clone https://github.com/anjupriya-v/e-Learning-spot.git
```
- Redirect to the cloned repo directory

- Open up the terminal

- Install the dependencies
```
npm install
```
- Start the project
```
npm start
```
